Questions

After completing the lab recap as a class the following questions:

General Discussion Questions

Q: Is the manifest uploaded with the application bits?
A: No. It is parsed by the cf CLI, then the cf CLI invokes specific Cloud Controller endpoints.

Questions From the Lab

Q: Assuming you have application running with a bound service (MySQL). You push the application again with a manifest, but said service is not specified in the manifest. Is the service still bound to the application?
A: Yes. The end state is an accumulation of all commands run against the app.

Q: What are some cf commands that don’t have to be explicitly made when using manifests?
A: cf bind-service, cf set-env
